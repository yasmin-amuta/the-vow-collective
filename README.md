# The Vow Collective — Wedding Photography Portfolio

A refined, animation-led portfolio website for The Vow Collective, a wedding photography studio.
The site focuses on emotion, pacing, and visual storytelling — combining cinematic media, subtle motion, and an editorial layout to reflect the studio’s evocative and intimate brand identity.

Built with performance and responsiveness in mind while maintaining a high-end, Awwwards-inspired aesthetic.

---

## Tech Stack

- **HTML5** — semantic structure
- **CSS3** — responsive layouts, custom cursor, transitions
- **JavaScript (ES6)** — interactions and logic
- **Lenis Scroll** — smooth scrolling experience
- **GSAP** — animations, timelines, ScrollTrigger
- **GSAP SplitText** — animated typography

---

## Features

- Responsive layout optimized for desktop, tablet, and mobile
- Cinematic intro video with animated split-text logo
- Smooth scroll experience with GSAP ScrollSmoother
- Scroll-triggered image and text animations
- Custom animated cursor with magnetic hover effects
- Animated full-screen navigation drawer
- Subtle editorial hover interactions and parallax effects
